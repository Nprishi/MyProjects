# Auth Starter - Production Ready

Complete authentication system with Next.js 15, TypeScript, Tailwind CSS

## Features

✅ Next.js 15 App Router  
✅ TypeScript (Strict Mode)  
✅ JWT Authentication  
✅ Protected Routes  
✅ Form Validation (Zod)  
✅ State Management (Zustand)  
✅ Server State (TanStack Query)  
✅ Dark Mode Support  
✅ Responsive Design  
✅ Production Ready

## Setup

```bash
cd auth-starter
npm install
npm run dev
```

## Scripts

- `npm run dev` - Dev server
- `npm run build` - Build
- `npm start` - Production
- `npm run lint` - Lint
- `npm run format` - Format
- `npm run test` - Tests

## Architecture

- Feature-based structure
- Atomic design components
- Zustand for state
- TanStack Query for data
- Axios with interceptors
- Clean separation of concerns
